# 🧰 Development Tools Guide

> Tools required to learn and build applications using ASP.NET Core Web API, SQL Server, and React.

---

## 💻 Backend Development (.NET)

### IDE / Code Editor Visual Studio = Everything-in-one tool to build, debug, and run .NET applications easily
- Visual Studio (Recommended for beginners & enterprise apps)
- Visual Studio Code (Lightweight and flexible)

### Runtime & SDK
- .NET SDK (Required to build and run Web API projects)

### API Testing Tools
- Swagger (Built-in API testing UI)
- Postman (External API testing tool)

---

## 🗄️ Database (SQL Server)

### Database Engine
- Microsoft SQL Server

### Database Management Tools
- SQL Server Management Studio (SSMS)
- Azure Data Studio (Optional lightweight alternative)

---

## ⚛️ Frontend Development (React)

### Runtime
- Node.js

### Package Managers
- npm (default)
- yarn (optional)
- pnpm (optional)

### Build Tools
- Vite (Recommended)
- Create React App (optional)

### Browser
- Google Chrome (for development & debugging)

## 🚀 Recommended Stack (Best Practice)

### Backend
- ASP.NET Core Web API
- Dapper (with Stored Procedures)
- JWT Authentication

### Database
- SQL Server (schema-based multi-tenant)

### Frontend
- React + Vite
- Context API (Auth + Global State)

---

## 💡 Notes

- Prefer Stored Procedures for business logic
- Keep controllers thin and services clean
- Follow consistent project structure
- Focus on real-world projects for learning

---

## ⭐ Tip

> Tools are important, but real learning comes from building projects.

Build:
- CRUD APIs
- Admin panels
- Multi-tenant SaaS apps
